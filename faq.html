I have an issue. Where can I report it?

If you have an issue please first check if someone else have the same issue as you. If not then create a new issue at the Issue Tracker.

Why are some of the qualities not appearing?

This is caused by YouTube limiting the availability of qualities when Dash Playback is disabled.

When will the next version get released?

A new version will be released when I think it's ready to be released. If you're impatient you can help test the developer version of YouTube Center.
