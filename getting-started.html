Install

To install YouTube Center please select the correct version below.

Userscript

For Greasemonkey or Tampermonkey.

Chrome/Blink based browsers

For the Chrome browser and other Blink based browsers including Opera (v15 or newer).

Get help to install on Chrome here.

Firefox

Native add-on for Firefox.

Install on Userscript
Install on Chrome Add-on page
Install on Firefox Add-on page

Maxthon

The extension for the Maxthon browser.

Safari

For the Safari browser.

Old Opera

For the older version of Opera (v14 or less).

Install on Maxthon Add-on page
Install on Safari
Install on Opera

What's next?

After you have successfully installed YouTube Center you will have to go to YouTube and a dialog will appear which will show you how to access the settings. From there you can configure YouTube Center just how you want it.
